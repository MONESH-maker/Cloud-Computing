## The objective of this task was to create and configure a cloud storage system using Supabase and to understand how data can be stored, managed, and accessed through the cloud. Supabase was used as the cloud platform due to its simplicity and ease of use, especially for beginners.

Initially, a new project was created in Supabase by providing the required details such as project name, database password, and region selection. After the project setup was completed, the storage service was accessed from the Supabase dashboard. A new storage bucket was then created, which acts as a container for storing files in the cloud. The bucket was configured as a public bucket to allow easy access to the stored files.

Next, sample files including a PDF document, an image file, and a text file were uploaded into the bucket. These files were stored securely within the cloud storage system. After uploading, public URLs were generated for each file, which allowed them to be accessed directly through a web browser.

The functionality of the storage system was verified by opening the file URLs in the browser. The successful display of files confirmed that the bucket was properly configured and that access permissions were correctly set.

Through this task, important concepts such as cloud storage, buckets, file management, and public access permissions were clearly understood. The task was successfully completed by implementing a working cloud storage solution using Supabase.
